## 💡 Instalación
***
1. Clonar el proyecto 
- git clone https://github.com/ImmerseTechHub/ImmerseTechHub.git

2. Instalar las dependencias de node con el comando:
- npm install

3. Para que el proyecto inicie:
- npm start

4. Para correr los test:
- npm test
